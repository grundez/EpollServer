# Описание
Сервер работает по принципу механизма мультиплексирования, используя Epoll API.
На вход принимаются сообщения клиента, если сообщение начинается с символа / - оно интерпретируется как команда, в противном случае зеркалируется клиенту.

### Команды
Сервер поддерживает следующие команды:
* **/time** — вернуть текущего времени и даты
* **/stats** — узнать статистику подключенных клиентов
* **/disconnect** — отключиться от сервера
* **/shutdown** — завершить работу сервера

### Работа с программой
**Запуск сервера**
```cmd
chmod -x build.sh
./build.sh
./build/EpollServer/EpollServer
```
