# Описание
Сервер работает по принципу механизма мультиплексирования, используя Epoll API.
На вход принимаются сообщения клиента, если сообщение начинается с символа /, то оно интерпретируется как команда. В противном случае зеркалируется клиенту.

### Команды
Сервер поддерживает следующие команды:
* **/time** — возврат текущего времени и даты
* **/stats** — возврат статистики подключенных клиентов
* **/shutdown** — завершение работы
